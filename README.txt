最小構成 Streamlit Cloud版

GitHubへアップロードするのは、このフォルダの中身だけです。
不要なbatファイルは入っていません。

必要ファイル:
- streamlit_app.py
- requirements.txt
- settings.yaml
- core/
- data/
- output/

Streamlit Cloudの Main file path は以下です。
streamlit_app.py

使い方:
1. GitHubリポジトリを空にする、または新規作成
2. このフォルダの中身をすべてアップロード
3. Streamlit CloudでDeploy
4. 画面左の条件を変えて「この条件でスクリーニング実行」
